# Thalamus - How to Use

## Getting Started

Thalamus is a graph-based visualization and editing tool for creating interconnected concept maps.

## Editor Basics

### Creating Nodes

- Press **N** to add a new node at the center of the canvas
- Click the **+** button in the left toolbar

### Editing Nodes

- **Double-click** a node to start editing the title
- Press **Enter** to confirm and move to body text (if expanded)
- Press **Escape** to cancel editing
- Click **"+ Add notes"** on a selected node to add body text

### Text Formatting

Both title and body text support rich text formatting:

- Select text to reveal the floating toolbar
- **Bold**: Click the B button or press **Cmd+B** (Ctrl+B on Windows)
- **Italic**: Click the I button or press **Cmd+I** (Ctrl+I on Windows)
- **Underline**: Click the U button or press **Cmd+U** (Ctrl+U on Windows)

### Connecting Nodes

- Drag from a node's handle (small circle) to another node to create a connection
- Click an edge to select it and customize its appearance

### Navigation

- **Cmd+K** (Ctrl+K on Windows) to open search
- Use the **Focus** button to zoom in on a selected node and its connections

## Keyboard Shortcuts

| Shortcut     | Action                           |
| ------------ | -------------------------------- |
| N            | Add new node                     |
| Cmd/Ctrl + K | Open search                      |
| Cmd/Ctrl + S | Save graph                       |
| Cmd/Ctrl + B | Bold text                        |
| Cmd/Ctrl + I | Italic text                      |
| Cmd/Ctrl + U | Underline text                   |
| Enter        | Confirm node title               |
| Escape       | Cancel editing / Exit focus mode |

## Styling

### Node Styling

Select a node to access the Node Style inspector on the right panel:

- **Color**: Change the node background color
- **Shape**: Rounded, Pill, Circle, or Square
- **Size**: Small, Medium, or Large
- **Text Colors**: Customize title and body text colors separately
- **Edge Padding**: Adjust spacing around connection handles

### Edge Styling

Select an edge to access the Relationship inspector:

- **Color**: Change the edge color
- **Thickness**: Adjust line width
- **Style**: Solid or Dashed
- **Curvature**: Bezier, Smoothstep, or Straight
- **Markers**: Arrow styles for start and end

### Map Style

Click the paintbrush icon in the left toolbar to apply styles to all nodes at once.

### Auto Layout

Click the grid icon in the left toolbar to automatically arrange nodes using different layout algorithms.

## Saving and Sharing

- Sign in to enable cloud saves
- Graphs auto-save after 3 seconds of inactivity
- Use the Share button to generate a shareable link
